# e04 - 注音文解碼器

讓 AI 看懂用英文鍵盤打出來的注音文。

## 為什麼需要這個技能？

台灣人偶爾會用「注音文」溝通 — 也就是直接用英文鍵盤的按鍵來表達注音符號。例如：

- `cl3` = ㄏㄠˇ = **好**
- `su3` = ㄋㄧˇ = **你**
- `su3cl3` = **你好**

沒有這個技能，AI 會把這些當成無意義的英文字母。有了它，AI 就能自動辨識並解碼。

## 使用場景

### 場景 1：解碼注音文

```
你：cl3 是什麼意思？
AI：「cl3」→ ㄏㄠˇ → 好
```

### 場景 2：混合中英文的注音文

```
你：我今天 cp3 累
AI：「cp3」→ ㄏㄣˇ → 很
    → 我今天很累
```
