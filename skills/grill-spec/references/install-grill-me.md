# Installing grill-me

grill-spec requires the [grill-me](https://github.com/mattpocock/skills) skill. Two installation paths:

## Option A: Using find-skills (if available)

If the find-skills skill is installed, invoke it to search for and install grill-me:

```
Use find-skills to install "grill-me"
```

## Option B: Manual installation

```bash
npx skills add mattpocock/skills/grill-me
```

Or clone manually into your agent's skills directory.
